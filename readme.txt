A custom Gutenberg block to embed Google Maps.

== Description ==

Google Maps Gutenberg Block is a custom block for Gutenberg editor. With this block, users can embed Google Maps into their posts and pages without leaving the editor.

= Features =

* Easily embed Google Maps.
* Set width, height, zoom level, and more for the map.
* No need to mess with API keys inside the editor; set it once in the plugin settings.

== Installation ==

1. Upload `google-maps-gutenberg-block` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to 'Google Maps Gutenberg' under the Settings menu and set your Google Maps API key.
4. Use the new block in the Gutenberg editor.

== Frequently Asked Questions ==

= Where do I get a Google Maps API Key? =

You can obtain an API key from [Google Cloud Console](https://console.cloud.google.com/).

= I set the API key, but the map doesn't show up. Why? =

Ensure that the API key is correct and that the Google Maps Embed API is enabled in your Google Cloud Console.
